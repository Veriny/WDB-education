Gitlet

Gitlet is a subset of the Git version control system implemented in Java. It uses a tree backend to store commits and support basic commands such as add, commit, push, and pull.

Gitlet 2

Gitlet 2 improves upon Gitlet 1 by adding a GUI, which can display the commit tree as well as free the user from using git CLI commands.

Gitlet 3

Gitlet 3 improves upon Gitlet 2 by adding the ability to add quantum remotes, i.e. the ability to add remotes across parallel universes. By making a decision, there will exist a multiverse which you can pull from where you made the opposite.

Gitlet 4

Gitlet 4 improves upon Gitlet 1 by adding the ability to connect their git repositories with a real bonsai tree, which would grow in accordance to how the commit tree grew.

Gitlet 5

Gitlet 5 umproves upon Gitlet 4 by connecting to the Josh Hug hivemind API to criticize commit messages that are made. It incorporates Gitlet 3's quantum remotes to access APIs across multiverses.

Gitlet 6

Gitlet 6 improves upon Gitlet 5 by cutting all the unnecessary features added in iterations 3-5, meaning that Gitlet 6 is just the same thing as Gitlet 2.
